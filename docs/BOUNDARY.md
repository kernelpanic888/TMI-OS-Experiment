# Sandbox Boundary

Status:

```text
local sandbox
not a publication layer
not a proof claim
not a private-secrets store
```

Allowed:

- small `.i1` sketches;
- local visual experiments;
- draft formulas;
- guard tests;
- examples that can be deleted or promoted later.

Not allowed:

- API keys, tokens, passwords, private keys;
- private passports;
- work/client data;
- claims of biological life, legal identity, or empirical physics proof;
- automatic publication;
- links to closed repositories as public claims.

Promotion rule:

```text
sandbox artifact
-> guard-check
-> review
-> only then promote to TLFL / docs / publication package
```
